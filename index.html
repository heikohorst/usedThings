Usedthings

HowTo organize "Super"-markets for "used things"

Download this project as a .zip file Download this project as a tar.gz file

Abstract

This Guide and Software will help you to organize a market to sell a high number of things which are good enough for second use, like clothes, toys, games, books, bicycles etc..

This market is organized like a department store, items are sorted by type and sizes. All Items have price tags, no contact between buyer and vendor.

A number of helpers need to sort out, overview the market, cash items and sort the rest back to the vendors boxes. ( Usually these helpers can be recruited from the vendors )

Software

  • label printing for item preparation
  • cash register
  • disbursement procedure ,-printouts

Features

  • fast keypad entry (no pointer device required)
  • barcode scanner enabled (HID/Keyboard emulation)
  • easy sale setup
    • market context ( motto, date, duration )
    • data entry ( required article data fields, barcode content )
    • tax/donation calculation ( price surcharge or revenue deduction )
    • seller/payee registration
    • article pre registration optional ( for price / article cross checks at register )
  • item-label- / receipts- / disbursement-printout ( PDF mail-out ?)

Architecture

  • easy server setup ( config-less )
  • zero-installation client (Html/Javascript)
  • cooperative access (no restricted users)
  • transaction journal/ no "deletes"
  • redundant/client replicated data storage with offline capability
